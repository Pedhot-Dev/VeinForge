# VeinForge (Migrated)

VeinForge is a migrated version of **Mighty Miner**, now updated for Minecraft **1.21.11**.

## Project Status & Warning

**⚠️ MESSY CODE & EXPERIMENTAL STATE**

This version is a result of a major migration from the original Mighty Miner (1.8.9). During this process:
- Significant portions of the code were refactored or removed.
- The codebase is currently "messy" and contains dead or non-functional code.
- **Do not expect most features to work.** Many legacy features are broken or partially implemented.

This project was updated strictly for personal needs. Several features, such as the **Fishing Macro**, were added as "flukes" to assist with personal goals (like leveling up). 

## What's Changed

This project has undergone a significant evolution from its initial migration:

- **Config System Migration**: Originally using OneConfig, the project migrated to **MoulConfig** (Modern) because OneConfig was not suitable for Minecraft 1.21. This required custom implementations for HUD elements, route HUDs, and other UI components.
- **Mapping Migration**: The project initially used **Yarn** mappings. However, to ensure better compatibility and stability, the entire codebase was refactored and migrated to **Official Mojang mappings**.
- **Feature Additions**: A **Fishing Macro** was added specifically to assist with personal goals (like leveling up). While experimental, it serves its purpose for the current developer.
- **Current Stability**: Although the project contains many legacy features and experimental additions, the **Commission Macro** is the only feature that is currently thoroughly tested and confirmed to work reliably. Other features may be non-functional or in a "messy" state due to the migration.
