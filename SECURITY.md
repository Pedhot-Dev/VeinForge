# Security Policy

## Supported Versions

We currently support the following versions of VeinForge:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue. Instead, please report it privately by contacting the maintainers.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
